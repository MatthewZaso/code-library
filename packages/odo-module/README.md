# Odo Module

> Augment a class' static methods with helpers

## Install

```shell
npm install @odopod/odo-module --save
```

## Quick Start

```js
import OdoModule from '@odopod/odo-module';

class CoolModule {}

CoolModule.Classes = {
  BASE: 'cool-module',
};

OdoModule.register(CoolModule);
```
