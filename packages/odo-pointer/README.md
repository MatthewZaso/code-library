# Odo Pointer

> An abstraction for pointer, mouse, and touch events.

## Install

```shell
npm install @odopod/odo-pointer --save
```

## Quick Start

```js
import OdoPointer from '@odopod/odo-pointer';

const pointer = new OdoPointer(document.querySelector('.js-my-sidebar'));
```
