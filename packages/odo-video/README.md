# Odo Video

> A lightweight html5 video wrapper.

## Install

```shell
npm install @odopod/odo-video --save
```

## Quick Start

```js
import OdoVideo from '@odopod/odo-video';

const player = new OdoVideo(document.getElementById('video-wrapper'));
```
