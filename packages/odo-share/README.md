# Odo Share

> Share links to social media.

## Install

```shell
npm install @odopod/odo-share --save
```

## Quick Start

```html
<a href="#" class="odo-share" data-service="twitter">Share on Twitter</a>
```

or

```js
import OdoShare from '@odopod/odo-share';

OdoShare.share({
  service: 'twitter'
});
```
