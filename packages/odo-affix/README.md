# Odo Affix

> Makes an element fixed position while its within a container.

## Install

```shell
npm install @odopod/odo-affix --save
```

## Quick Start

```js
import OdoAffix from '@odopod/odo-affix';

const affix = new OdoAffix(document.querySelector('.js-my-sidebar'));
```
