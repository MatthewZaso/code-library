# Odo Tabs

> A component for switching between panes of content.

## Install

```shell
npm install @odopod/odo-tabs --save
```

## Quick Start

```js
import OdoTabs from '@odopod/odo-tabs';

const tabs = new OdoTabs(document.querySelector('.js-my-sidebar'));
```
